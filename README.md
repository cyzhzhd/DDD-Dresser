# Dresser

## 기능

- 카테고리 별 최저가격 브랜드와 상품 가격, 총액을 조회하는 API
- 단일 브랜드로 모든 카테고리 상품을 구매할 때 최저가격에 판매하는 브랜드와 카테고리의 상품가격, 총액을 조회하는 API
- 카테고리 이름으로 최저, 최고 가격 브랜드와 상품 가격을 조회하는 API
- 브랜드 및 상품을 추가 / 업데이트 / 삭제하는 API
